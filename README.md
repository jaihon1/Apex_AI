# Apex_AI

### Apex Legends
A character based First Person Shooter online game.
Battleroyal: 20 teams of 3.
https://www.ea.com/games/apex-legends

### Program
Apex_AI is a program that learns how to play Apex Legends 

## Goal
  * Have Fun.
  * Train single agent to play Apex Legends (Agent + Human + Human).
  * Group single agents together to work in team (Agent_0 + Agent_1 + Agent_2).

## Concepts
  * Convolutional Neural Network
  * Reinforcment Learning (Q-Learning)

## Tools
  * Python3
  * OpenCV
  * TensorFlow
  * Apex Legends (Windows)
